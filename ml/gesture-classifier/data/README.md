# Local motion data

Recorder exports, processed windows, and model files stay local and are ignored
by Git. Use pseudonymous subject IDs such as `P001`; never put a participant's
name in a filename or label.

- `raw/`: CSV and manifest files exported by the browser recorder.
- `processed/`: optional derived tables.
- `artifacts/`: trained `.skops` files and model cards.

Share a dataset only after participants have consented and identifying notes
have been removed.
